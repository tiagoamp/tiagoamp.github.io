Project SIBE-REQUER

DESCRIPTION:
Project goal was to build an application to manage the lifecycle of requests for public services (assistential and retirement) in government agencies.
When our team was assigned to this project, it was in progress and several months delayed. Stakeholders was displeased. Requirements was gathered and fixed in contract some years before, and changes wasn't allowed. Codebase was over engineered. There was no shippable product after more than one year of project.
Project was cancelled after negociation between our dev team, PMO and stakeholders, since its purpose was outdated. We convinced PMO to use an MVP approach to a new version of the project.
DURATION:
05 months (Start date: 03/Aug/2009 - End date: 31/Dec/2009).
ROLE:
Developer / Team Leader
RESPONSIBILITIES:
Java implementation; Project status report; Scope management.
TECH & TOOLS:
Java, EJB, JSF, JPA, Oracle Database.
ACHIEVEMENTS:
Negotiate with PMO, clients and key stakeholders to abort the project and change to an MVP (Minimum Viable Product) approach. No more efforts and budget spent with outdated requirements, saving money and time of development (about $7000 savings per month in dev effort).

Project SIRC version 1

DESCRIPTION:
Project for development of a system from scratch to get population data from public notaries all over the country.
These people's personal data would be used for census, reports, statistics and to support government decisions.
There are about 8 thousand notaries offices which send daily data to SIRC, like birth, marriage and death registrations. These data may be sent by web services or web-forms provided by SIRC application, or even loading data in a SIRC desktop application that stores and send the records later.
Project should build about 6 (web/batch/desktop) modules to handle all this data, planned in monthly releases.
DURATION:
19 months (Start date: 12/Mar/2010 - End date: 21/Oct/2011).
ROLE:
Developer / Team Leader.
RESPONSIBILITIES:
Java implementation; Project status report; Resource planning; Scope management; Deployment monitoring.
TECH & TOOLS:
Java, EJB, JSF, JPA, Web Services(Soap-Apache CXF), Maven, Oracle Database.
ACHIEVEMENTS:
Deployed a robust application, highly accessed with minimal downtime. Risk reducing using monthly monitored deployment.
Developed the product with a newly hired employees, coaching them to become a self-organized team within few weeks.
URLs:
  • Portal: http://www.sirc.gov.br/

  • Desktop application (download): http://www.sirc.gov.br/static/downloads/sirc_cartorio.zip
